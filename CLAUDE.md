# Not the canonical GrowMatic SA website

**Per the site owner's explicit instruction:** this Lovable-built repo
(`growmaticsa-com`) is **not** what he considers "the website." The
canonical, live site is the separate `growmatic-sa` repo (hand-built
static HTML/CSS/JS, referred to by him as "the website I made with
Claude design").

This repo may look nearly identical (similar branding/copy) and may
independently be configured as a GitHub Pages custom domain for
`growmaticsa.com` — if so, that is a source of domain contention with
`growmatic-sa` (inconsistent DNS/HTTPS behavior, links "not opening").
The intended fix is to remove/change this repo's custom-domain binding
in GitHub Settings > Pages, not to treat this repo as primary.

Do not default to editing this repo when asked to make changes to "the
website," "growmaticsa.com," or similar — check with the user or assume
`growmatic-sa` unless told explicitly to work here.

This repo remains a legitimate, separately-maintained property (still
built/deployed, e.g. for experimentation or as a secondary reference) —
just not the one the owner treats as authoritative.
